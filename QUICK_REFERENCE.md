# 📖 Quick Reference Guide\n\nCommon tasks and solutions for portfolio development.\n\n## Running the Site\n\n### Development Server\n```bash\nnpm run dev\n```\nOpen http://localhost:5173 in your browser. Hot reload on file changes.\n\n### Production Build\n```bash\nnpm run build\n```\nCreates optimized `dist/` folder ready to deploy.\n\n### Preview Production Build\n```bash\nnpm run preview\n```\nOpen http://localhost:4173 to preview production build locally.\n\n## Adding Content\n\n### Update Hero Section\nFile: `src/components/Hero.jsx`\n```jsx\n// Change these values:\nconst name = 'JOSHUA G. ESCALA'; // Your name\nconst title = 'Web Developer'; // Your title\nconst tagline = '...'; // Your tagline\n```\n\n### Add a New Project\nFile: `src/components/Projects.jsx`\n```jsx\nconst projects = [\n  {\n    title: 'My Awesome Project',\n    description: 'What this project does.',\n    tags: ['React', 'Node.js', 'MongoDB'],\n    link: 'https://live-demo-link.com',\n    github: 'https://github.com/username/project-name',\n  },\n  // ... more projects\n];\n```\n\n### Update Skills\nFile: `src/components/ProfileSummary.jsx`\n```jsx\nconst skills = ['HTML', 'CSS', 'JavaScript', 'React', 'Node.js'];\n// Edit the array to match your skills\n```\n\n### Update Contact Info\nFile: `src/components/Contact.jsx`\n```jsx\n// Update these values:\nemail: 'your-email@example.com'\nphone: '+1 (555) 123-4567'\nlocation: 'Your City, Country'\nlinkedin: 'https://linkedin.com/in/yourprofile'\ngithub: 'https://github.com/yourname'\n```\n\n### Add Education Entry\nFile: `src/components/EducationExperience.jsx`\n```jsx\nconst timeline = [\n  {\n    year: '2023 - Present',\n    title: 'Your Degree',\n    organization: 'University Name',\n    description: 'Brief description.',\n  },\n  // ... more entries\n];\n```\n\n### Add Certificate Image\n1. Place image file in `src/assets/certificates/`\n   - Supported: JPG, PNG, JPEG\n   - Recommended size: 800px width\n2. Update `src/components/Certificates.jsx` with matching filename:\n   ```jsx\n   image: getImage('your-certificate-image.jpg'),\n   ```\n\n## Customizing Styles\n\n### Change Theme Colors\nFile: `src/index.css`\n```css\n:root {\n  --bg: #0f172a;           /* Main background */\n  --surface: #0b1220;      /* Card backgrounds */\n  --text: #e6eef8;         /* Text color */\n  --accent: #2563eb;       /* Primary color (blue) */\n  --accent-2: #93c5fd;     /* Secondary color */\n  --muted: #94a3b8;        /* Muted text */\n}\n```\n\n### Style a Component\nEach component has a `.module.css` file:\n```\nsrc/components/ComponentName.jsx      // Component logic\nsrc/components/ComponentName.module.css // Component styles\n```\n\nEdit the CSS file to customize appearance:\n```css\n.heading {\n  font-size: 2rem;\n  color: var(--accent-2);\n  margin-bottom: 2rem;\n}\n```\n\n## Troubleshooting\n\n### Images Not Showing\n**Symptom**: Images appear as colored boxes or don't load\n\n**Solution**:\n1. Verify file is in `src/assets/` or `src/assets/certificates/`\n2. Check filename matches exactly (case-sensitive)\n3. Restart dev server: Kill terminal, run `npm run dev` again\n4. Clear browser cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)\n\n### Build Errors\n**Symptom**: `npm run build` fails with errors\n\n**Solution**:\n1. Check terminal output for error details\n2. Verify syntax (missing semicolons, brackets)\n3. Run `npm install` to update dependencies\n4. Delete `node_modules/` and run `npm install` again\n\n### Styles Not Applying\n**Symptom**: CSS changes don't show up\n\n**Solution**:\n1. Hard refresh browser: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)\n2. Clear browser cache\n3. Restart dev server\n4. Check that you're editing the `.module.css` file, not another CSS\n\n### Site Takes Too Long to Load\n**Symptom**: Page loads slowly\n\n**Solution**:\n1. Optimize images: Use [TinyPNG](https://tinypng.com)\n2. Resize images to web size (800px width max)\n3. Use WebP format for modern browsers\n4. Check performance: [pagespeed.web.dev](https://pagespeed.web.dev)\n\n## File Structure Reminder\n\n```\nsrc/\n├── assets/               # Images and static files\n│   ├── certificates/     # Certificate images\n│   ├── profile.jpg       # Hero profile photo\n│   └── react.svg\n├── components/           # Reusable components\n│   ├── Navbar.jsx\n│   ├── Hero.jsx\n│   ├── ProfileSummary.jsx\n│   ├── Services.jsx\n│   ├── EducationExperience.jsx\n│   ├── Certificates.jsx\n│   ├── Projects.jsx\n│   ├── Contact.jsx\n│   ├── Footer.jsx\n│   ├── Button.jsx\n│   └── *.module.css      # Component styles\n├── pages/\n│   └── Home.jsx          # Main page\n├── App.jsx               # Root component\n├── main.jsx              # Entry point\n├── index.css             # Global styles\n└── App.css               # App-level styles\n```\n\n## Dependencies\n\n- **react** (18.2.0): UI library\n- **react-dom** (18.2.0): React DOM rendering\n- **react-router-dom** (6.14.2): Client-side routing\n- **framer-motion** (10.13.0): Animation library\n- **react-icons** (4.11.0): Icon library\n\nAdd new dependencies:\n```bash\nnpm install package-name\n```\n\n## Browser Support\n\nThe portfolio works on:\n- Chrome 90+\n- Firefox 88+\n- Safari 14+\n- Edge 90+\n\nTest in multiple browsers before deploying!\n\n## Performance Tips\n\n1. **Optimize Images**\n   - Resize to web size (max 800px width)\n   - Compress with TinyPNG\n   - Use WebP format where possible\n\n2. **Lazy Load Images**\n   - Use `loading=\"lazy\"` attribute\n   - Defer below-fold images\n\n3. **Minimize Code**\n   - Use `npm run build` for production\n   - Vite automatically minifies and bundles\n\n4. **Cache Control**\n   - Static files cached by browser\n   - Set long cache expiration for Vercel/Netlify\n\n## Accessibility Checklist\n\n- [ ] All buttons have keyboard focus visible\n- [ ] All images have alt text\n- [ ] Form labels are associated with inputs\n- [ ] Color contrast passes WCAG AA (4.5:1)\n- [ ] No text relies on color alone\n- [ ] Site is navigable with keyboard only\n- [ ] Animations can be paused (prefers-reduced-motion)\n- [ ] Semantic HTML is used (nav, main, footer, etc.)\n\n## Useful Links\n\n- [React Docs](https://react.dev)\n- [Vite Docs](https://vitejs.dev)\n- [React Router](https://reactrouter.com)\n- [Framer Motion](https://www.framer.com/motion)\n- [React Icons](https://react-icons.github.io/react-icons/)\n- [CSS Variables Reference](https://developer.mozilla.org/en-US/docs/Web/CSS/--*)\n- [Web Accessibility](https://www.w3.org/WAI/)\n\n---\n\n💡 **Pro Tip**: Keep this file handy during development for quick reference!\n"
